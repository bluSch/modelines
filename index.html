Modelines for Sublime Text

Currently in development.

Modelines for sublime text will provide modeline support for Sublime Text compatible with Vim1, and later Emacs. I'm also considering parsing hashbang lines to determine filetype.

My desire for this plugin comes from one specific file I deal with very frequently and so might be a bit of an overreaction. That file, the Vagrantfile, is used by Vagrant, and has no extension but is a ruby file. Other files that I can see benefiting from proper modeline support would include template files or other files with nonsensical or nonexistant extensions.

1The vim support is based on vim's source and as such will provide support for for ex, vi, and elvis syntax in addition to vim's).
